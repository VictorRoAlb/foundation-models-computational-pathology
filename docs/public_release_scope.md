# Public Release Scope

This repository is intentionally centered on the evaluation and reporting structure of the thesis project.

## Included

- documentation of datasets, models and evaluation settings;
- safe reporting scripts;
- synthetic examples and example configuration files;
- aggregated figures and notebook templates.

## Excluded

- clinical data;
- raw pathology reports;
- private model checkpoints;
- restricted model weights;
- embedding dumps;
- spreadsheets and local execution artifacts from the private thesis workspace.

## Why this balance was chosen

The most valuable public contribution of the project is the reproducible evaluation structure: how the benchmarks are organized, how retrieval metrics are summarized and how final comparative figures are built. That can be shared responsibly without publishing any sensitive material.
